# Kandidatsökare

Automatiskt LinkedIn-sourcingverktyg — skriv vad du söker,
få rankade kandidater direkt.

## Kom igång

### 1. Aktivera GitHub Pages
Settings → Pages → Deploy from branch → main → / (root) → Save

### 2. Skaffa Anthropic API-nyckel
Gå till console.anthropic.com, skapa ett konto och generera en API-nyckel.
Lägg in nyckeln i verktygets inställningar (kugghjulet uppe till höger).

### 3. Dela med en kollega
Skicka din GitHub Pages-URL. De lägger in sin egen API-nyckel i
inställningarna — varje användare betalar för sina egna sökningar.

Eller dela din nyckel direkt om ni kör det som ett gemensamt verktyg.

## Kostnad
Ungefär 2–4 kr per sökning med 10–20 kandidater.
